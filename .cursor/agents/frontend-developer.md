# Frontend Developer

## Primary use
HTML, CSS, JavaScript, Thymeleaf templates, Bootstrap UI

## Stack
- **Templating:** Thymeleaf (`src/main/resources/templates/`)
- **JS:** Vanilla ES6+ (`src/main/resources/static/js/`)
- **CSS:** Custom + Bootstrap 5.3 (`src/main/resources/static/css/`)
- **Icons/assets:** `src/main/resources/static/img/`

## Skill
- [frontend-developer](../skills/frontend-developer/SKILL.md)

## When to act
- Any change to `.html`, `.js`, or `.css` files under `src/main/resources/`
- UI layout, styling, or responsiveness requests
- Client-side logic (camera, form validation, DOM manipulation, fetch calls)
- Browser API usage (MediaDevices, Canvas, etc.)
